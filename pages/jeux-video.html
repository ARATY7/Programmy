La Programmation de jeux vidéo

N’importe quel jeu vidéo d’aujourd’hui est codé dans un langage de programmation comme le C++, C#, le Java, … Tous ces langages ont leurs points forts comme faibles et c’est pour cela que tous les jeux sont codés avec plusieurs langages. Un jeu bien connu aujourd’hui, Minecraft, lui est codé entièrement en Java par exemple.

Les jeux vidéo sont construits comme les sites internet, plusieurs fichiers de code fonctionnant ensemble. Pour débuter dans la programmation de jeux vidéo, nous pouvons apprendre le langage Scratch qui peut aider à comprendre le fonctionnement de langage plus complexe.

Source ludosciences.blogspot.com

C'est un langage simple permettant de faire avancer, reculer, tourner le chat selon les instructions qu'on lui donne. C'est une initiation à la programmation de jeux vidéo.

Nous utilisons des langages tels que le C++, le C#, le Java, Lua, etc. pour programmer des jeux d'aujourd'hui. Pour programmer des jeux comme Fortnite ou Red dead Redemption, il faut des heures et des heures de programmation effectuées par des passionnés. Les premiers jeux étaient développés en langage assembleur permettant le développement rapide. Sur les tout premiers ordinateurs, le BASIC était le langage utilisé avant la sortie du C.

Source fr.vecteezy.com

Les jeux vidéo ne sont pas tous portables, les développeurs ont dû trouver des moyens de porter leurs jeux sur les ordinateurs comme sur les consoles. Depuis 1980, les ordinateurs et les consoles étaient super différents ce qui compliquait la portabilité des jeux entre les différents appareils.
